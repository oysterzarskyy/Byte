# Byte Mobile

Byte is now being shaped as an Android-first mobile app prototype for a voxel-style experience.

## What’s included

- Android app shell with a main menu style landing screen
- Terrain generation using a lightweight noise-based chunk generator
- A core Kotlin module for world generation logic
- GitHub automation and project scaffolding

## Build

```bash
./gradlew :app:assembleDebug
```

## Run

Open the project in Android Studio or install the generated APK on an Android device.
